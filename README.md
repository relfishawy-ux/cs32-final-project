# Autonomous Drone Pathfinding Simulator

## Project Description
This project focuses on designing a Python-based simulation of an autonomous drone that can navigate from a starting point to a destination while avoiding obstacles. The core of the project is implementing and comparing pathfinding algorithms that allow the drone to make intelligent decisions in a constrained environment.

The drone operates in a 2D grid-based map where obstacles, boundaries, and target locations are defined. The system will compute safe and efficient paths and simulate how the drone would move step-by-step through the environment.

## Goals
- Implement pathfinding algorithms such as BFS, DFS, Dijkstra, or A*
- Simulate a drone navigating through an obstacle-filled environment
- Ensure the drone avoids collisions and respects movement constraints
- Compare efficiency of different algorithms (path length, computation time)
- Visualize or print the computed path in a clear and interpretable way

## Why This Project
This project allows me to explore computational problem-solving through real-world-inspired applications. Autonomous navigation is a fundamental problem in robotics, and this project focuses on the algorithmic side of that challenge. By working in simulation, I can focus on decision-making, optimization, and system design without being limited by hardware constraints. Additionally, I get to test the code on the drone I built for one of my EE projects before!

## Expected Outcome
A Python program that:
- Takes a map with obstacles, a start point, and a goal
- Computes an optimal or near-optimal path
- Simulates the drone’s movement along that path
- Optionally compares multiple algorithms and evaluates their performance
